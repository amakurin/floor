Механизм работы

По понятным причинам, данный документ не описывает всех алгоритмов и тонкостей, но дает общее понимание для интересующегося пользователя. Детально о механизме робота можно узнать в личном общении с его создателями. Войти в контакт с создателями можно отправив электронное письмо по адресу creators@floor16.ru

1. Как работает поиск

1.1. Робот регулярно опрашивает основные открытые площадки объявлений и социальные сети на предмет новых объявлений о сдаче в аренду жилой недвижимости.
1.2. Как только обнаружено новое объявление, Робот добавляет его в индекс.
1.3. Добавленные в индекс объявления в течение 1-5 минут проходят обработку, которая состоит из следующих этапов:
  • извлечение данных из текста объявления;
  • бинарная классификация объявления по признаку агент\собственник;
  • если объявление не опознано в качестве агентского – публикация объявления на сайте.
1.4. После публикации объявления некоторое время повторно классифицируются, и могут быть сняты с публикации, если Робот выносит по ним иной вердикт.
1.5. Несколько раз в сутки Робот отслеживает снятие объявлений с публикации на стороне источников, что позволяет в результатах поиска выдавать только актуальные на текущий момент объявления. Снятые с публикации объявления по-прежнему остаются доступны пользователям по прямым ссылкам. На страницах таких объявлений выводится соответствующее сообщение.
1.6. Все решения Робота о публикации и снятии с публикации объявлений синхронизируются с содержимым пабликов ВКонтакте в течение 10-20 минут. Это означает, что информация опубликованная Роботом в ВК также актуализирована. На данный момент в пабликах выкладываются только объявления с фотографиями. По общему решению пользователей Робот может быть перенастроен. Пользователи могут выражать свои мысли по этому поводу в соответствующих обсуждениях ВК.
